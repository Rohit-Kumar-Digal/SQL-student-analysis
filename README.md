# 📊 SQL Student Performance Analysis

## 📌 Objective

Analyze student data using SQL to identify trends and insights.

## 🔍 Analysis

* Student distribution by city and gender
* Average marks by subject
* Top-performing students
* Attendance vs performance
* Monthly trends

## 📈 Key Insights

* Higher attendance leads to better performance
* Some cities perform better
* Few students are below average
* Subject performance varies

## 🛠 Tools Used

* SQL (MySQL / SQL Server)
